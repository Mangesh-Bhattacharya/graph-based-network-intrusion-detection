# Security Policy

## Supported Versions
This project is an academic research project for the course *Complex Networks*.  
It is not intended for production use, but we still welcome responsible reporting of security issues.

| Version | Supported |
|---------|-----------|
| main    | Yes       |

## Reporting a Vulnerability
If you discover a vulnerability, please follow responsible disclosure:

1. Do **not** open a public GitHub issue.
2. Contact the team privately at:
   - jothi.jayaraman@ontariotechu.net
   - mangesh.bhattacharya@ontariotechu.net
   - salmaanafroz.kuthpudeen@ontariotechu.net
   - vishnuprasath.sathyanarayanan@ontariotechu.net
     
4. Provide:
   - Description of the issue
   - Steps to reproduce
   - Potential impact

We will review and respond within **7 days**.

## Scope
This project includes:
- Graph construction scripts
- ML/GNN models
- Jupyter notebooks
- Dataset preprocessing code

This project does **not** include:
- Production deployment
- Real‑time IDS systems
- Cloud or network infrastructure

## Responsible Use
This project is for **educational and research purposes only**.  
Do not use it for unauthorized network scanning or intrusion activities.
